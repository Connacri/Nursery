## ✨ Features by Role

### 👑 Super Admin
- 🏢 **Multi-tenant management**: Create, suspend, and migrate tenants.
- 📈 **Business Intelligence**: MRR, ARR, Churn, and Global Occupancy.

### 👩‍🏫 Educator & 🩺 Medical Staff
- 📝 **Intelligent Journal**: Rapid entry, photo sharing, and daily summaries.
- 💉 **Health Records**: Vaccinations, Allergies, and Growth curves.

### 👨‍👩‍👧 Parent / Guardian
- 📱 **Real-time Feed**: Activities, Meals, and Nap times.
- 📄 **Document Vault**: Contracts and Invoices in one place.

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Connacri/Nursery.git
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the app**
   ```bash
   flutter run
   ```
