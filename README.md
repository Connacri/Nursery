## ✨ Features by Role

### 👑 Super Admin
- 🏢 **Multi-tenant management**: Create, suspend, and migrate tenants.
- 📈 **Business Intelligence**: MRR, ARR, Churn, and Global Occupancy.
- 🛠️ **Global Monitoring**: Real-time user tracking and API health.

### 🏫 Crèche Admin & Director
- 👥 **Operational Control**: Staff, Room, and Vehicle assignments.
- 💰 **Financial Suite**: Invoices, Expenses, and Cash flow tracking.
- 📢 **Communication**: Newsletters and Global announcements.

### 👩‍🏫 Educator & 🩺 Medical Staff
- 📝 **Intelligent Journal**: Rapid entry, photo sharing, and daily summaries.
- 💉 **Health Records**: Vaccinations, Allergies, and Growth curves.
- ⚠️ **Smart Alerts**: Medication reminders and prevention.

### 🚌 Driver & 📦 Stock Manager
- 🗺️ **Smart Logistics**: GPS tracking, boarding validation, and SOS.
- 🛒 **Inventory**: Automatic restocking alerts and purchase requests.

### 👨‍👩‍👧 Parent / Guardian
- 📱 **Real-time Feed**: Activities, Meals, and Nap times.
- 💳 **Marketplace**: Easy ordering and payments.
- 📄 **Document Vault**: Contracts and Invoices in one place.

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Connacri/Nursery.git
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the app**
   ```bash
   flutter run
   ```

## 🛠️ Architecture

Built with **Clean Architecture** and **BLoC Pattern** to ensure scalability, testability, and enterprise-grade performance.

- **Frontend**: Flutter (PWA, Mobile, Desktop)
- **Backend**: Supabase (Postgres, RLS)
- **Auth**: Firebase Authentication
- **Offline**: Intelligent data synchronization
