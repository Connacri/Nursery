# Palette's Journal

## 2024-05-22 - [Initial Setup]
**Learning:** Flutter applications often lack basic platform-specific UX refinements like autofill hints and password visibility toggles out of the box.
**Action:** Always check for text field refinements (autofill, keyboard types, visibility toggles) when encountering login forms.

## 2024-05-22 - [Dashboard Polish]
**Learning:** Adding hover effects and tooltips to administrative dashboards significantly improves the "pro" feel of an enterprise application without adding much complexity.
**Action:** Use AnimatedScale and MouseRegion for subtle, delightful desktop interactions.

## 2024-05-22 - [Build Configuration]
**Learning:** Modern Firebase plugins (Auth/Core) often require iOS 15.0+. Keeping the deployment target up to date is crucial for build stability.
**Action:** Always check IPHONEOS_DEPLOYMENT_TARGET when adding or updating Firebase dependencies.
